Player 1

  • You pick the sequence of colors for your opponent to guess.

Player 2

  • The objective of the game is to guess the exact positions of colors in the sequence.
  • A color can be used only once in a code sequence.
  • After filling a line with your guesses, the game responses with the result of your guess.
  • For each color in your guess that is in the correct color and correct position in the code sequence, the game displays a small red color.
  • For each color in your guess that is in the correct color but is NOT in the correct position in the code sequence, the game displays a small white color.
  • You win the game if you guess the exact sequence chosen by your opponent.
  • You lose the game if you use all attempts without guessing the actual sequence.
